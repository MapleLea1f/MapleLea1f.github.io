# 线段树

线段树（Segment Tree）是一种非常灵活的数据结构，它可以用来解决多种范围查询问题，例如在数组中查找区间和、区间最大值、区间最小值等。线段树是一种二叉树，每个节点代表一个区间[L, R]，其中L和R是数组的下标。

### 基本概念

1. **节点**：每个节点代表一个区间[L, R]。
2. **叶子节点**：叶子节点代表单个元素，即区间[i, i]。
3. **内部节点**：内部节点代表的区间是其左右子节点代表区间的并集。

### 结构

线段树的基本结构如下：

- 根节点代表整个区间[0, N-1]。
- 每个非叶子节点有两个子节点，左子节点代表区间[L, (L+R)/2]，右子节点代表区间[(L+R)/2+1, R]。

### 操作

线段树支持以下几种基本操作：

1. **构建（Build）**：从数组构建线段树。
2. **查询（Query）**：查询某个区间的信息，例如区间和、区间最大值等。
3. **更新（Update）**：更新数组中的某个元素，并相应地更新线段树。

### 示例

假设我们有一个数组`arr = [1, 3, 5, 7, 9, 11]`，我们希望构建一个线段树来查询区间和。

#### 构建线段树

```python
class SegmentTree:
    def __init__(self, arr):
        self.n = len(arr)
        self.tree = [0] * (4 * self.n)  # 线段树数组，大小为4*n
        self.build(arr, 0, 0, self.n - 1)

    def build(self, arr, node, start, end):
        if start == end:
            self.tree[node] = arr[start]
        else:
            mid = (start + end) // 2
            self.build(arr, 2 * node + 1, start, mid)
            self.build(arr, 2 * node + 2, mid + 1, end)
            self.tree[node] = self.tree[2 * node + 1] + self.tree[2 * node + 2]

    def query(self, node, start, end, l, r):
        if r < start or end < l:
            return 0
        if l <= start and end <= r:
            return self.tree[node]
        mid = (start + end) // 2
        left_sum = self.query(2 * node + 1, start, mid, l, r)
        right_sum = self.query(2 * node + 2, mid + 1, end, l, r)
        return left_sum + right_sum

    def update(self, node, start, end, idx, val):
        if start == end:
            self.tree[node] = val
        else:
            mid = (start + end) // 2
            if start <= idx <= mid:
                self.update(2 * node + 1, start, mid, idx, val)
            else:
                self.update(2 * node + 2, mid + 1, end, idx, val)
            self.tree[node] = self.tree[2 * node + 1] + self.tree[2 * node + 2]

# 示例使用
arr = [1, 3, 5, 7, 9, 11]
seg_tree = SegmentTree(arr)
print(seg_tree.query(0, 0, len(arr) - 1, 1, 3))  # 查询区间[1, 3]的和，输出15
seg_tree.update(0, 0, len(arr) - 1, 2, 6)  # 更新数组元素arr[2]为6
print(seg_tree.query(0, 0, len(arr) - 1, 1, 3))  # 再次查询区间[1, 3]的和，输出16
```
