# 变长数组+哈希表

变长数组（Dynamic Array）和哈希表（Hash Table）是两种常见的数据结构。

### 变长数组（Dynamic Array）

变长数组，也称为动态数组或可变数组，是一种在运行时可以动态调整大小的数组。与静态数组不同，静态数组的大小在编译时就已经确定，而变长数组的大小可以在程序运行过程中根据需要进行扩展或缩减。

#### 特点：

1. **动态大小**：可以在运行时增加或减少数组的大小。
2. **随机访问**：可以通过索引快速访问元素。
3. **内存管理**：通常需要额外的内存管理，以确保在扩展或缩减时不会浪费内存。

#### 实现：

在许多编程语言中，变长数组是通过底层的数据结构实现的，例如在C++中使用`std::vector`，在Python中使用`list`。

```python
# Python 示例
dynamic_array = []
dynamic_array.append(1)  # 添加元素
dynamic_array.append(2)
dynamic_array.append(3)

for element in dynamic_array:
    print(element, end=" ")
```



### 哈希表（Hash Table）

哈希表是一种使用哈希函数将键（key）映射到值（value）的数据结构。它通过哈希函数计算出一个索引，然后将键值对存储在该索引位置，从而实现快速的插入、删除和查找操作。

#### 特点：

1. **快速查找**：平均情况下，插入、删除和查找操作的时间复杂度为O(1)。
2. **哈希冲突**：不同的键可能映射到相同的索引位置，需要解决哈希冲突（例如使用链表或开放寻址法）。
3. **动态大小**：可以动态调整大小以保持性能。

#### 实现：

在许多编程语言中，哈希表是通过内置的数据结构实现的，例如在C++中使用`std::unordered_map`，在Python中使用`dict`。

```python
# Python 示例
hash_table = {}
hash_table["one"] = 1  # 插入键值对
hash_table["two"] = 2
hash_table["three"] = 3

for key, value in hash_table.items():
    print(f"{key}: {value}")
```

### 例题

实现RandomizedSet 类：

- RandomizedSet() 初始化 RandomizedSet 对象
- bool insert(int val) 当元素 val 不存在时，向集合中插入该项，并返回 true ；否则，返回 false 。
- bool remove(int val) 当元素 val 存在时，从集合中移除该项，并返回 true ；否则，返回 false 。
- int getRandom() 随机返回现有集合中的一项（测试用例保证调用此方法时集合中至少存在一个元素）。每个元素应该有 相同的概率 被返回。

你必须实现类的所有函数，并满足每个函数的 平均 时间复杂度为 O(1) 。

### 题解

```python
class RandomizedSet:

    def __init__(self):
        self.nums=[]
        self.indices={}

    def insert(self, val: int) -> bool:
        if val in self.indices:
            return False
        self.indices[val]=len(self.nums)
        self.nums.append(val)
        return True

    def remove(self, val: int) -> bool:
        if val not in self.indices:
            return False
        id=self.indices[val] # 获取val在可变列表中的索引值
        self.nums[id]=self.nums[-1] # 将可变列表中的val修改为可变列表中的最后一个的值
        self.indices[self.nums[id]]=id
        self.nums.pop()
        del self.indices[val]
        return True



    def getRandom(self) -> int:
        return choice(self.nums)
```
